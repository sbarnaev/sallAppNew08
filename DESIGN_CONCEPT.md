# Дизайн-концепция: Premium Glassmorphism для САЛ ПРОФИ

## Философия дизайна

**Quiet Luxury для профессионального инструмента**

Интерфейс должен ощущаться как дорогой личный инструмент мастера — спокойный, надёжный, помогающий думать, а не отвлекающий. Каждый экран отвечает на один вопрос мастера.

---

## Визуальная система

### Glassmorphism как материал интерфейса

**Принципы:**
- Полупрозрачные карточки с blur 20–30px
- Тонкие светлые границы (rgba(255,255,255,0.6))
- Чёткое ощущение слоёв и глубины
- Активные элементы визуально «ближе» (больше opacity, больше blur)
- Вторичные элементы «дальше» (меньше opacity, меньше blur)

**Технические параметры:**
- Backdrop blur: 20–30px
- Background: rgba(255,255,255,0.4–0.7)
- Border: 1px solid rgba(255,255,255,0.6)
- Shadow: мягкие, многослойные, с лёгким цветным оттенком

### Цветовая палитра

**Основа:**
- Фон: молочный, тёплый серый (#FAF9F7, #F5F4F2)
- Текст: тёмно-серый (#1A1A1A, #2D2D2D)
- Вторичный текст: средний серый (#6B6B6B, #8B8B8B)

**Акцент (один спокойный холодный цвет):**
- Основной акцент: приглушённый сине-серый (#4A6FA5, #5B7FB8)
- Акцент светлый: #E8EDF5
- Акцент тёмный: #3A5A8A

**Запрещено:**
- Яркие цвета (красный, зелёный, фиолетовый как основные)
- Градиенты на элементах интерфейса
- Множество цветов одновременно

### Типографика

**Шрифт:**
- Inter / SF Pro / Manrope (современный гротеск)
- Чёткая иерархия размеров
- Минимум bold (только заголовки и акценты)
- Воздух и читабельность важнее эффектов

**Иерархия:**
- H1: 28–32px, font-weight: 600
- H2: 22–24px, font-weight: 600
- H3: 18–20px, font-weight: 500
- Body: 15–16px, font-weight: 400
- Small: 13–14px, font-weight: 400

### Компоновка

**Mobile-first:**
- Основное использование с телефона
- Минимум действий на экране
- Центр внимания: текущий клиент / расчёт / консультация
- Остальное вторично

**Принципы:**
- Меньше блоков, больше смысла
- Карточки строго по функциям
- Один экран = один вопрос мастера

## Структура экранов

### Mobile (320–768px)

**Навигация:**
- Боковое меню (drawer) с glassmorphism
- Иконки без текста (экономия места)
- Активный пункт: более яркий, ближе к зрителю

**Dashboard:**
- Одна большая карточка с текущим фокусом (последний клиент/расчёт)
- Статистика: компактные glass-карточки в одну колонку
- Быстрые действия: крупные кнопки, одна колонка

**Списки (клиенты, расчёты, консультации):**
- Карточки на всю ширину
- Минимум информации на карточке
- Поиск: стеклянная панель сверху

**Детали:**
- Полноэкранный режим
- Крупные, читаемые блоки
- Действия: фиксированная панель снизу

### Desktop (768px+)

**Навигация:**
- Боковая панель: узкая (64px), расширяется при hover (240px)
- Glassmorphism на панели
- Плавные переходы

**Dashboard:**
- Сетка 2–3 колонки
- Больше пространства для информации
- Статистика: горизонтальные карточки

**Списки:**
- Сетка 2–3 колонки
- Больше деталей на карточках
- Фильтры: боковая панель или верхняя панель

**Детали:**
- Центрированный контент (max-width: 1200px)
- Боковые панели для дополнительной информации
- Действия: верхняя панель

## Компоненты

### Glass Card
- Полупрозрачный фон
- Blur 20–30px
- Тонкая светлая граница
- Мягкая тень
- Hover: увеличение opacity и blur

### Glass Button
- Полупрозрачный фон
- Тонкая граница
- Минимальный blur (10–15px)
- Акцентный цвет только на hover/active
- Без ярких цветов по умолчанию

### Glass Input
- Полупрозрачный фон
- Blur 15–20px
- Тонкая граница
- Focus: увеличение opacity, акцентная граница

### Glass Navigation
- Полупрозрачная панель
- Blur 25–30px
- Активный пункт: более яркий, ближе
- Плавные переходы
